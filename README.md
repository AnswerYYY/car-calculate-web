# Car Calculator (Vue 3 + Vite)

移动端购车成本计算器，基于 Vue 3 与 Vite。

## 启动方式

```bash
npm install
npm run dev
```

## 构建与预览

```bash
npm run build
npm run preview
```

## 工程目录

```text
car-calculator/
├─ index.html
├─ package.json
├─ vite.config.js
└─ src/
   ├─ App.vue
   ├─ main.js
   ├─ assets/
   │  └─ styles/
   │     └─ main.css
   ├─ constants/
   │  └─ defaults.js
   └─ utils/
      └─ calculator.js
```
